# Cleaned Datasets

This directory contains the datasets that have been processed and cleaned following the Data Science Methodology.

## Contents

- `__init__.py`: Makes this directory a Python package for easy data loading.
- Cleaned versions of datasets from the `drafts/datasets/` folder will be saved here once the cleaning scripts are executed.

## Methodology

1. **Data Collection**: Raw data is gathered in `drafts/datasets/`.
2. **Data Preparation**: Cleaning, transformation, and feature engineering.
3. **Data Storage**: Processed data is stored here for modeling and analysis.
