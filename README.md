# Data Science Methodology Project

This project focuses on applying the Data Science Methodology to diverse datasets, with a particular focus on personality traits, productivity, and health metrics.

## Workspace Structure

- [cleaned-datasets/](cleaned-datasets/): Processed data and Python package.
- [drafts/](drafts/): Raw datasets, experiments, and methodology notes.
- [plots/](plots/): Generated visualizations and figures.
- [oad_big_five.ipynb](oad_big_five.ipynb): Big Five personality dataset analysis.
- `req.txt`: Required dependencies.

## Methodology

1. **Business Understanding**: Define goals for each dataset.
2. **Data Understanding**: Exploratory Data Analysis.
3. **Data Preparation**: Cleaning and feature engineering.
4. **Modeling**: Predictive and descriptive models.
5. **Evaluation**: Assessing performance and insights.
6. **Deployment & Feedback** (Pending).
